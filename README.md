## Overview

**Architecta** is a cross-platform project scaffolding software that allows developers, students, and teams to generate **production-ready project structures** with **one click**. It supports multiple languages and frameworks and follows industry best practices for folder organization, starter files, and boilerplate code.

Architecta is lightweight, offline-first, and fully customizable.

---

## Features

* **Dynamic Languages & Frameworks**
  Python (Flask, FastAPI, CLI), JavaScript (Node.js, Express), Bash automation
* **Template-Based Generation**
  JSON templates define folder structures, files, and starter code
* **Frontend GUI**
  Browser-based interface, works offline
* **API Endpoint**
  `/generate` endpoint exposed via FastAPI for programmatic generation
* **One-Click Executable**
  Can be packaged into an exe for Windows or a binary for Linux/macOS
* **Offline & Secure**
  No cloud dependencies, no data collection, runs locally

## Installation

1. **Clone repository**

```bash
git clone https://github.com/yourusername/architecta.git
cd architecta
```

2. **Create Python virtual environment**

```bash
python -m venv venv
source venv/bin/activate    # Linux/macOS
venv\Scripts\activate       # Windows
```

3. **Install dependencies**

```bash
pip install fastapi uvicorn
pip install "fastapi[all]"  # for CORS
```

---

## Running Locally (Development Mode)

1. **Start backend**

```bash
uvicorn backend.app.main:app --reload
```

2. **Open frontend**

Open `frontend/index.html` in your browser.

3. **Generate project**

* Fill form: Project Name, Language, Framework, Base Path
* Click **Generate Project**
* Project folder will appear at the specified path

---

## One-Click Executable

To create a desktop-like application:

1. **Install PyInstaller**

```bash
pip install pyinstaller
```

2. **Package into executable**

```bash
pyinstaller --onefile --add-data "frontend;frontend" run_app.py
```

3. **Run the executable**

* Windows: `dist/run_app.exe`
* Linux/macOS: `dist/run_app`

‚úÖ Opens frontend automatically and runs backend API.

## Template Guidelines

* **Folder Structure:**
  Defined in `folders` array in `template.json`
* **Files & Content:**
  Defined in `files` dictionary in `template.json`
* **Extending Templates:**
  Add a new language/framework folder under `templates/` with `template.json`

## Example Template

`templates/python/flask/template.json`:

```json
{
  "meta": {
    "language": "python",
    "framework": "flask",
    "description": "Flask project scaffold",
    "version": "1.0"
  },
  "structure": {
    "folders": ["app", "app/routes", "app/services", "tests", "docs"],
    "files": {
      "app/__init__.py": "",
      "app/main.py": "from flask import Flask\napp = Flask(__name__)\n@app.route('/')\ndef home():\n    return 'Hello from Architecta!'",
      "requirements.txt": "flask\n",
      "README.md": "# Flask Project Generated by Architecta\n"
    }
  }
}
```
